## О чём проект

**Balkanski kod** — это игра-викторина, где знакомые мелодии звучат на сербском языке. Вы слушаете фрагмент, узнаёте мелодию и пытаетесь вспомнить или угадать название. Вместо зубрёжки — музыка, эмоции и постепенное погружение в язык.

> *«Как бы звучали известные песни, если бы их создали в Сербии?»*

### Как играть

1. 🎧 **Слушаешь** — знакомая мелодия, но на сербском
2. ✍️ **Угадываешь** — вводишь название трека
3. 💡 **Подсказки** — если застрял, есть подсказки от простой к явной
4. 📻 **Радио** — фоновое прослушивание всех треков для погружения

## Что внутри

### 🧩 Викторина
- 14 треков с 5 уровнями сложности
- Умная проверка ответов (fuzzy matching, транслитерация, расстояние Левенштейна)
- Прогресс сохраняется в localStorage
- Статистика: попытки, подсказки, процент угаданных

### 📻 Балканское радио
- Случайный порядок треков, без перемотки — как настоящее радио
- Автопауза при начале игры
- Лимит 30 минут прослушивания
- Ссылки на оригиналы в SUNO

### 🌍 4 языка интерфейса
- Русский, English, Srpski (latinica), Српски (ћирилица)
- Динамический контент страниц на каждом языке

### ⭐ Поддержка проекта
- Донаты через Telegram Stars
- Telegram-бот принимает Stars-платежи
- На сайте — ненавязчивые кнопки с deep link на бота

## Технологии

### Frontend
- **React 18** + **TypeScript 5.9** + **MUI 7** + **Vite 5**
- **React Context + useReducer** — стор прогресса
- **i18next** — интернационализация
- **React Router 6** — HashRouter для GitHub Pages
- **Feature-Sliced Design** — архитектура слоями

### Backend (бот донатов)
- **Cloudflare Workers** — serverless, бесплатный хостинг
- **grammY** — Telegram Bot API
- **Telegram Stars** — приём платежей

### Инструменты
- **Vitest** — 117 тестов
- **Steiger** — проверка FSD архитектуры
- **CLI Wizard** (PHP + OpenAI) — генерация треков и подсказок
- **ESLint** — линтинг

## Быстрый старт

```bash
# Клонирование и установка
git clone https://github.com/Mikeloangel/balkanskikod-quiz.git
cd balkanskikod-quiz
npm install

# Настройка
cp .env.example .env

# Запуск
npm run dev
```

### Команды

```bash
npm run dev            # Dev server :5173
npm run build          # Production сборка (тесты + линт + tsc + vite)
npm test               # Запуск тестов
npm run lint           # ESLint
npm run lint:fsd       # Проверка FSD архитектуры
npm run deploy         # Деплой на GitHub Pages
npm run bot:deploy     # Деплой бота на Cloudflare Workers
```

## Почему музыка работает для изучения языка

Когда слышишь знакомую мелодию на новом языке — мозг уже знает контекст. Не нужно переводить каждое слово, смысл приходит интуитивно. А радио фоном создаёт постоянный контакт с языком без усилий.

**Пример:**

> 🇷🇺 «С любимыми не расставайтесь»
>
> 🇷🇸 «Не растајте се с вољенима»

Смысл тот же, но конструкция естественная для сербского языка.

## Как добавить трек

```bash
# Интерактивный мастер — генерирует подсказки через OpenAI
php cli/wizard.php
```

Или вручную: добавить запись в `src/shared/config/tracks.ts` и MP3 в `public/tracks/`.
