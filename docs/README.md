# Документация Balkanski kod

## Обзор

**Balkanski kod** — музыкальная квиз-игра на угадывание балканских мелодий с интегрированным радио плеером. Проект построен на React + TypeScript с использованием Material-UI и архитектурой Feature-Sliced Design.

## Технологический стек

- **Frontend**: React 18 + TypeScript
- **UI**: Material-UI (MUI)
- **Роутинг**: React Router (HashRouter для GitHub Pages)
- **Состояние**: Zustand + React Contexts
- **Хранение**: localStorage
- **Сборка**: Vite
- **Тесты**: Vitest + React Testing Library

## Архитектура

### Feature-Sliced Design
```
src/
├── shared/     # Общий код (lib, config, contexts, models)
├── entities/   # Сущности (progress)
├── features/   # Фичи (отсутствуют в текущей версии)
├── widgets/    # Виджеты (radioPlayer)
├── pages/      # Страницы (home, track, about)
└── app/        # Приложение (App.tsx, routing)
```

### Контекстная архитектура
- **AudioContext** — управление игровым аудио плеером
- **RadioProvider** — управление радио плеером
- **UIDialogsContext** — управление модальными окнами
- **TrackNavigationContext** — навигация между треками
- **TrackGameUIContext** — UI логика игры

## Ключевые особенности

### Система проверки ответов
- Многоуровневая проверка: прямое совпадение → набор слов → нечеткое сравнение
- Поддержка русских и оригинальных названий
- Автоматическая транслитерация кириллицы
- Учет опечаток и словоформ

### Радио плеер
- Независимая работа в фоне
- Случайный порядок треков
- Автоматическая пауза при запуске игры
- Ограничение времени прослушивания (30 минут)
- Интеграция с Suno

### Прогресс и статистика
- Хранение в localStorage
- Подробная история попыток
- Глобальная статистика решения
- Поддержка сброса прогресса

## Разработка

### Правила разработки
См. [`AGENTS.md`](../AGENTS.md) для подробных правил разработки и архитектурных ограничений.

### Тестирование
```bash
npm test              # Все тесты (122 теста)
npm run test:ui       # UI режим Vitest
npm run test:coverage # Покрытие кода
```

### Линтинг и сборка
```bash
npm run lint          # ESLint
npm run lint:fsd      # Проверка FSD архитектуры
npm run build         # Production сборка
npm run deploy        # Деплой на GitHub Pages
```

## Деплой

Проект развернут на GitHub Pages с использованием HashRouter для корректной работы навигации.

### Переменные окружения
- `VITE_APP_NAME` — название приложения
- `VITE_APP_BASE_URL` — базовый URL для шаринга
- `VITE_RADIO_MAX_PLAY_TIME_MINUTES` — лимит радио (по умолчанию 30)
