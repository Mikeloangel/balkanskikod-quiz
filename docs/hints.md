# Универсальный промпт для генерации 1 трека

Ты помогаешь заполнить **один объект трека** для файла `src/shared/config/tracks.ts`.
Сербский кириллица

## Цель
Сгенерировать качественный объект `Track`, который:
- легко вставляется в TypeScript-массив без ручной чистки;
- не ломает игровую механику;
- дает полезные, но не спойлерные подсказки.

## Контекст проекта
- аудитория: русскоязычные пользователи;
- формат: игра “угадай название трека”;
- язык интерфейса: русский;
- `safe` не должен явно палить финальный ответ.

## Строгие правила
1. Возвращай **один** объект трека в формате TypeScript.
2. Не меняй `id`, `links.local` и `dates.added`, если они уже заданы.
3. `links.local` всегда: `'/tracks/<id>.mp3'`.
4. `links.suno`: валидная ссылка или пустая строка `''`.
5. `names`:
   - `safe`: коротко, атмосферно, без прямого спойлера;
   - `serbian`: естественный перевод/адаптация; если невозможно, аккуратная транслитерация;
   - `russian`: каноничное название на русском;
   - `original`: на языке оригинала (если оригинал русский, можно дублировать `russian`).
6. `hints`: 2-4 штуки, по возрастанию явности.
7. Подсказки:
   - про название и узнаваемые образы;
   - без полного пересказа истории песни;
   - без прямого спойлера в первой подсказке.
8. `difficulty`: целое число 1-5.
9. Используй одинарные кавычки и trailing comma как в проекте.

## Критерии качества hints
- Hint 1: широкий культурный контекст/атмосфера.
- Hint 2: конкретнее по образу или ключевому слову.
- Hint 3-4 (если есть): почти подводит к названию, но не называет его дословно.

## Формат ответа (обязательный)
1) Короткое пояснение (3-6 буллетов): почему выбраны `safe`, `serbian`, `hints`, `difficulty`.
2) Затем **только один** код-блок с готовым объектом TypeScript.
3) Если есть спорные места, после кода дай 1-2 альтернативы (коротко).

## Шаблон объекта
```ts
{
  id: 'track-xxx',
  links: {
    local: '/tracks/track-xxx.mp3',
    suno: '',
  },
  names: {
    safe: '',
    serbian: '',
    russian: '',
    original: '',
  },
  hints: [
    '',
    '',
  ],
  difficulty: 3,
  dates: {
    added: 'YYYY-MM-DD',
  },
}
```

## Быстрая самопроверка перед отправкой ответа
- `safe` не спойлерит напрямую?
- `hints` реально помогают угадать название, а не сюжет?
- `difficulty` соответствует подсказкам?
- Объект валиден для прямой вставки в `tracks.ts`?
